SoilSense is an AI application developed to determine soil types and provide health insights using a machine learning model. It mainly targets small scale farmers, and it provides insights on soil types and steps to take toward maximizing crop yield and soil health. 

The main program runs identification through a CNN (convolutional neural network), so users have to upload an image that will be processed. This runs through the model, and the ouput is translated into insights for the user.
